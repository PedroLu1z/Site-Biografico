Quem foi James Joseph Sylvester?

James Joseph Sylvester foi um matemático inglês que contribui fundamentalmente no desenvolvimento da teoria matricial. Desempenhou papel fundamental no desenvolvimento da matemática nos Estados Unidos na segunda metade do século XIX.

O que é uma matriz?

Matriz podemos definir como uma estrutura matemática que organiza dados e informações em linhas e colunas, como uma tabela. São muitos utilizados em áreas do conhecimento, como a matemática, a informática e a gestão.

Características:

  • Os números que compõem uma matriz são chamados de elementos
  • Os elementos são identificados por colchetes, parênteses, barras simples ou barras duplas
  • As operações de adição, subtração e multiplicação são bem definidas
  • As matrizes são representadas da forma A mxn, onde M é o número de linhas e N é o número de colunas
  • Existem casos especiais de matrizes, como a matriz linha, a matriz coluna, a matriz quadrada, a matriz oposta e a matriz identidade

Aplicações

  • São utilizadas para acompanhar o progresso através de critérios, metas ou normas
  • São utilizadas para resolver problemas matemáticos de forma padronizada
  • São utilizadas para facilitar o trabalho com dados numéricos
  • São essenciais para o desenvolvimento da programação

Tipos de Matrizes:

Matriz identidade

Uma matriz quadrada A = {a ij}nxm definida por A = {a ij}nxm, onde aij = 1 <-> i = j e aij = 0 <-> i != j é denominada matriz identidade de ordem n

Matriz Quadrada

o número de linhas é igual ao número de colunas

2,1 2,2
